# Next Contact Form

This is my own app created in Next.js

## Technologies
- Next.js
- Tailwind CSS
- TypeScript
- ESLint
- Yup

## Before running the project:

Create .env file and paste

```
EMAIL_SERVICE=""
EMAIL_USER=""
EMAIL_PASS=""
EMAIL_PORT=
DISCORD_WEBHOOK_URL=""
```

Replace variables' values with your own values :)

Run the project by:
`yarn run dev`
or 
`yarn build && yarn start`
